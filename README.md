# Work Day Scheduler 

## Description
Schedule tasks throughout the day with this work day scheduler!
The scheduler dynamically refreshes to display the current day and colours events based on the time of day.

## Technologies 
* HTML
* CSS
* JavaScript
* jQuery 
* Bootstrap
* Day.js
